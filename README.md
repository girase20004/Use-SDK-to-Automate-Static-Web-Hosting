# Use-SDK-to-Automate-Static-Web-Hosting
Automated static website hosting on AWS S3 using Python and boto3. Uploads website files programmatically and enables hosting with minimal manual steps.

## 🎯 Purpose
Upload website to S3 using Python automation.

## 🧰 AWS Services Used
- Amazon S3
- boto3 (Python SDK)

## 📌 Project Overview
This project automates static website hosting on AWS S3 using Python. 
It uploads files, configures permissions, and enables hosting using boto3.

## 🚀 Features
- Automated deployment
- Easy file upload
- Static hosting
- Reduced manual work
